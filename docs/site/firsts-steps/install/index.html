    Preparando o ambiente

    Você pode instalar o Pix Verifier de duas formas:

    1. Clonando o respositório no Github e configurando o servidor e cada serviço manualmente, nginx, php, python... Sem dúvidas é a forma mais complicada (e dolorosa) de se fazer isso.

    2. Rodando via container, disponibilizado no Docker Hub, ou você mesmo buildando através do Dockerfile disponível no repositório.

    Nessa documentação cobriremos a forma mais simples que é baixando a imagem do Docker Hub e subindo os serviços.
